📌 TaskSPHERE App
A simple MERN stack task management application where users can sign up, log in, and manage tasks efficiently.

🚀 Features
✅ User Authentication (Signup & Login)
✅ Create, Edit, and Delete Tasks
✅ Task Status Management
✅ User Dashboard
